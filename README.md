## ✨ Features

- **🚀 Cinematic Splash Screen:** Modern glowing splash screen (`AuraFit Neo - Future of Motion Tracking`) on app launch.
- **📊 Interactive Dashboard:** Real-time tracking of step counts, active energy burned, and daily goals.
- **📈 Weekly Analytics Chart:** Visual weekly bar breakdown (Monday–Sunday) tracking performance consistency.
- **✍️ Manual Activity Logging:** Quick modal dialog to input custom exercise types, workout durations (minutes), and calories burned.
- **📝 Workout History List:** Displays recent activity logs ("Today's Workouts") dynamically linked to the database.
- **🎨 Glassmorphic Dark UI:** Premium neon-accented dark theme optimized for clarity and reduced eye strain.
